# 베르와 함께하는 로또 여행 로또 1등 갈끄니까

## 도메인 기능목록

- 구입 금액
    - [x] 구입금액은 천원 단위여야한다
    - [x] 구입금액은 음수가 아니여야한다.

- [ ] 10만원 이상은 사지못한다(현실반영 베르의 경험)

- 로또 숫자
    - [x] 로또 숫자는 1 이상 45 이하 숫자이다.

- 로또 숫자 발급기
    - [x] 1 이상 45 이하 무작위 숫자를 생성한다.

- 로또 발급기
    - [ ] 6개의 로또 번호를 발급한다.


- 로또
  - [x] 로또는 LottoNumber 6개로 이루어진 리스트를 가지고있다 
  - [x] 로또 숫자는 중복되면 안된다.

- 당첨 로또
    - [ ] 메인 번호와 보너스 번호가 있다.

- 은행
    - [ ] 당첨 등수를 확인한다.
    - [ ] 당첨 금액의 합계 계산한다.
    - [ ] 수익률을 계산한다.
